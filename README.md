# MAT3123

🎬 영화 리뷰 감성 분석 프로젝트
1. 프로젝트 목적

본 프로젝트는 수업 시간에 배운 머신러닝 및 딥러닝 이론을 실제 텍스트 데이터에 적용하여,
영화 리뷰가 긍정적인지 부정적인지 자동으로 분류하는 감성 분석 모델을 설계하는 것을 목표로 한다.

특히 강의에서 다룬

Logistic Regression

신경망(MLP)

PyTorch 기반 모델 구현
을 실습 수준에서 통합적으로 적용하였다.

2. 데이터셋

네이버 영화 리뷰 감성 데이터(NSMC)

약 20만 개의 한국어 영화 리뷰

각 리뷰는 긍정(1) 또는 부정(0) 라벨을 가짐

본 프로젝트에서는 전체 데이터 중 일부를 샘플링하여 실험 설계를 진행하였다.

3. 데이터 전처리

Pandas를 이용해 데이터를 로드하고 결측치를 제거하였다.

텍스트 데이터를 토큰화한 후,

TF-IDF(Vector Space Model) 방식으로 벡터화하여
고차원의 수치 데이터로 변환하였다.

이를 통해 머신러닝 및 딥러닝 모델이 텍스트를 처리할 수 있도록 하였다.

4. 모델 설계
(1) Baseline 모델

Logistic Regression

목적: 복잡한 모델과 성능 비교를 위한 기준선 설정

(2) Advanced 모델

PyTorch 기반 MLP (Multi-Layer Perceptron)

입력층 → 은닉층(ReLU) → 출력층 구조

손실 함수: Cross Entropy Loss

옵티마이저: Adam
